# Home Assistant Add-on: Hailo-10H VLM Chat

_Interactive Vision Language Model chat powered by Hailo-10H AI accelerator._

![Supports aarch64 Architecture][aarch64-shield]

## About

This add-on demonstrates real-time visual question answering using the
Hailo-10H AI accelerator on a Raspberry Pi 5. Point a camera at a scene,
capture a frame, and ask questions about what the AI sees.

It is designed as a **community blueprint** for building Hailo-10H powered
add-ons for Home Assistant.
